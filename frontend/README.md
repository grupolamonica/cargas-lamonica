# Frontend

Aplicacao React/Vite do projeto Lamonica.

## Responsabilidades

- Portal do motorista
- Painel administrativo
- Integracao com Supabase Auth e banco de dados
- Consumo da API local de rotas

## Estrutura principal

- `src/pages`: paginas da aplicacao
- `src/components`: componentes compartilhados e de UI
- `src/integrations`: clientes e tipos de integracoes externas
- `src/services`: servicos do frontend
- `src/test`: setup e testes do frontend
